# Planet Trivia

Juego de trivia sobre **planetas y exploración espacial**. Aplicación web ligera con preguntas interactivas y sistema de puntuación.

## Qué hace

- Presenta preguntas sobre planetas, astronomía y exploración espacial
- Sistema de puntuación por respuestas correctas
- Feedback visual inmediato al responder
- Interfaz responsive

## Stack

- HTML5 · CSS3 · JavaScript (vanilla)

## Uso local

```bash
git clone https://github.com/felixBD04/Planet-Trivia.git
cd Planet-Trivia
# Abre index.html en tu navegador
```

## Por qué este proyecto

Práctica de:

- Manejo de estado en aplicaciones interactivas simples
- Renderizado dinámico de contenido (preguntas, opciones, resultados)
- Lógica de validación de respuestas
- Diseño de UIs orientadas a juego sin frameworks
